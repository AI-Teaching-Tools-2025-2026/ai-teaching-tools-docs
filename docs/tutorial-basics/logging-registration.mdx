# Logging / Registration

Before accessing the platform, users must create an account. This ensures secure access and allows instructors to manage their content and tools effectively.

---
![Login/Registration Page](/img/login_page.png)

## Registering an Account

To get started, you will need to provide the following information:

- **Name**
- **Email Address**
- **Password**

### Password Requirements

For security purposes, your password must meet the following criteria:

- Must be **at least 8 characters long**
- Must contain **at least one uppercase letter**
- Must contain **at least one number**

:::tip Strong Password Tip
Using a mix of uppercase letters, lowercase letters, numbers, and symbols will make your account more secure.
:::

---

## Logging In

Once your account has been created, you can log in using your:

- **Email Address**
- **Password**

After successfully logging in, you will be redirected to the **home page**, where you can access all platform features, including:

- Question Bank  
- Quiz Builder  
- AI Tutor  

---

## Summary

- Registration is required before using the platform  
- Passwords must meet security requirements  
- After logging in, users gain full access to the application  

:::danger Important
Make sure to keep your login credentials secure and do not share your password with others.
:::
