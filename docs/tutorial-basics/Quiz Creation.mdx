# Quiz Creation

The **Quizzes** tab allows instructors to create, manage, and organize quizzes for their courses. From this page, you can create new quizzes, edit existing ones, delete quizzes, and filter through them efficiently.

---
![Quiz Creation Page](/img/quiz-builder.png)

## Quiz Management

When you navigate to the **Quizzes** tab, you will see a **quiz table** displaying all created quizzes.

### Table Information

Each quiz includes the following details:

- **Title**
- **Section**
- **Due Date**
- **Points**
- **Number of Questions**
- **Status**

### Actions

- Clicking on the **Title** allows you to preview the quiz from a student’s perspective  
- On the right side, you will find options to:
  - **Edit**
  - **Duplicate**
  - **Delete**

---

## Filtering Quizzes

You can filter quizzes to quickly find what you need. Available filters include:

- Title  
- Section  
- Due Date  
- Points  
- Number of Questions  
- Status  

:::tip
Use filters to efficiently manage large numbers of quizzes across different sections.
:::

---

## Creating a Quiz

To create a new quiz:

1. Navigate to the **Quizzes** tab  
2. Click **Create Quiz**  
3. Enter the following details:
   - **Quiz Title**
   - **Description**
   - **Section**
   - **Points**
   - **Due Date**

---

## Adding Questions

After entering quiz details, move to the **Questions** tab.

You have two options for adding questions:

### 1. Manual Creation

- Choose between:
  - **Multiple Choice**
  - **True / False**
- Add answer options by clicking **Add Answer**
- Save your changes after editing  

---

### 2. Question Bank (AI-Generated)

- Select questions from the **AI-generated question bank**  
- Questions are based on the uploaded textbook  
- Quickly populate quizzes with relevant content  

---

## Previewing and Finalizing

- You can **preview the quiz at any time** to see how students will view it  
- Once satisfied, click **Create Quiz** to finalize  

---

## Summary

- The Quizzes tab allows full quiz management (create, edit, delete, filter)  
- Quizzes can be built manually or using AI-generated question banks  
- Preview ensures accuracy before publishing  

:::danger Important
Make sure all questions and answers are correct before creating the quiz, as students will see the finalized version.
:::
