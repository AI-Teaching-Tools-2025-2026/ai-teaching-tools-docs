AI Teaching Tools - Main Features

Explore the core features of the AI Teaching Tools platform, including quiz generation, question banks, and an AI-powered tutor designed to support instructors and students.

📄️ Logging / Registration

Before accessing the platform, users must create an account. This ensures secure access and allows instructors to manage their content and tools effectively.

📄️ Course Creation

After logging in, instructors can create and manage courses. Each course acts as a central workspace for organizing textbooks, question banks, quizzes, and AI-powered tools.

📄️ Quiz Creation

The Quizzes tab allows instructors to create, manage, and organize quizzes for their courses. From this page, you can create new quizzes, edit existing ones, delete quizzes, and filter through them efficiently.

📄️ AI Tutor Creation

The AI Tutor is a course-aware chatbot powered by the textbook you uploaded. It allows instructors to create a customized teaching assistant that interacts with students based on course content and instructor-defined behavior.
